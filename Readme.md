# Facebook Messenger chatbot nodeJS
<br>
Extract the node_modules.rar<br>
# init:
<br>
`npm init`<br>
`npm install`<br>
`npm install --save express`<br>
`npm install --save request`<br>
`npm install --save body-parser`<br>
`npm i yandex-translate`<br>
`npm i cheerio`<br>

# heroku deployment:<br>
run heroku file or heroku.bat
